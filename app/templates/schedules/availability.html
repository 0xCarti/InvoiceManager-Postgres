Availability

Set the normal weekly pattern here, then use overrides for one-off exceptions.
How this works
Recurring availability is the user's normal weekly pattern, such as "Mondays 9:00 to 17:00". Overrides are date-specific exceptions, such as vacation, an appointment, or making someone available for a special event. If a user has no recurring windows yet, auto-assign treats them as generally available and only approved time off or unavailable overrides block the shift.
